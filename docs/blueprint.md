# **App Name**: FXPass Mobile

## Core Features:

- Secure Auth: User Registration & Authentication using mobile number and SMS OTP, following the Firebase Auth best practices.
- Wallet Dashboard: Display the user's PHP wallet balance and transaction history in a clean, intuitive interface.
- P2P Transfers: Facilitate peer-to-peer transfers using mobile number or QR code scanning.
- Cash Operations: Enable cash-in/cash-out functionality via InstaPay/PesoNet with clear instructions and status updates.
- AI KYC Submission Scanner: A tool to scan the contents of submitted KYC documentation using a GenAI model to compare to entered text, increasing the overall matching score, while surfacing potential discrepancies.
- FX Rate Display: Real-time KRW-to-PHP exchange rate powered by RedAppleX integration with fee calculation.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) for a modern, trustworthy feel, reminiscent of digital transactions and technological advancement.
- Background color: Light gray (#F0F0F0), providing a clean and neutral backdrop that allows the primary color and content to stand out.
- Accent color: Light Cyan (#E0FFFF) to provide subtle contrast, visually highlight interactive elements, and guide the user through key actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, providing a techy, scientific feel, combined with 'Inter' (sans-serif) for body text to ensure readability and a modern aesthetic.
- Crisp, minimalist icons to represent financial transactions, security features, and user profile elements.
- Clean, intuitive design with clear calls to action and easy navigation. Prioritize information hierarchy for effortless usability.
- Subtle, non-intrusive animations to provide feedback and enhance the user experience. For example, loading indicators during transactions and confirmations after successful actions.
